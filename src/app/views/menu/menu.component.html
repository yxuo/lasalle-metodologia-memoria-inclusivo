SENSORY RECALL

Bem-vindo ao Sensory Recall!

Este é um jogo de memória que foi projetado para ser jogado com um leitor de tela ou com o teclado.

Instruções:

  • Use o teclado numérico para selecionar as cartas.
  • Memorize as posições e tente encontrar os pares correspondentes.
  • O jogo oferece feedback audível para cada ação e combinação.
